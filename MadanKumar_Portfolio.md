### ✨ Background & Atmosphere Effects

```
1. PARTICLE WAVE MESH (Hero — most important element)
   - Flowing sine-wave made of tiny white dots (~1–2px)
   - Two waves: one left-bottom, one right-bottom
   - Dots animate slowly — wave flows continuously (loop)
   - Color: rgba(255,255,255,0.3–0.5) per dot
   - Libraries: tsParticles, three.js, or canvas animation
   - Effect: gives the feeling of an AI neural mesh / data flow

2. RADIAL GLOW HALOS
   - Soft white circular bloom on far left and far right edges
   - Position: fixed behind content, low opacity (0.04–0.06)
   - Size: 400–600px radius
   - Blends into background seamlessly (no hard edge)
   - Top-center: faint upward spotlight (like a stage light from below)

3. STAR FIELD / NOISE TEXTURE
   - Very subtle scattered white pixel dots across full bg
   - Opacity: 0.2–0.3 per dot
   - Gives "deep space" texture without overwhelming content

4. SECTION TRANSITIONS
   - Dark bg continues across all sections (no color breaks)
   - Sections separated only by vertical spacing, not color changes
   - Everything floats in the same dark cosmos
```

### 🎬 Animation Directives

```
HERO SECTION:
  - Background particle wave: continuous loop, slow drift (8–12s cycle)
  - Heading: fade-up on load (translateY 20px → 0, opacity 0 → 1, 0.6s ease-out)
  - Badge: fade-in 0.1s delay before heading
  - Subtitle: fade-up 0.15s after heading
  - CTA button: fade-up 0.2s after subtitle
  - Glow halos: slow pulse (scale 1 → 1.05 → 1, 4s loop)

SCROLL ANIMATIONS (Intersection Observer):
  - All sections: fade-up on enter viewport (translateY 30px → 0)
  - Stagger children: 0.1s delay between each card/element
  - Easing: cubic-bezier(0.22, 1, 0.36, 1) — fast out, slow settle

METRIC COUNTERS:
  - Numbers animate from 0 → final value on scroll into view
  - Duration: 1.5–2s, easeOutExpo curve
  - Examples: "0 → 72%", "0 → 40%", "0 → 74%"
  - Use CountUp.js or custom requestAnimationFrame loop

TESTIMONIAL CAROUSEL:
  - Fade transition between slides (not slide-swipe)
  - Auto-advance: 5s interval
  - Indicator dots: active = white filled, inactive = gray outline

NAVBAR:
  - On scroll: subtle blur + dark bg transition (backdrop-filter increases)

HOVER STATES:
  - Cards: border brightens (0.2s transition)
  - Buttons: brightness 1.1 + slight scale (1.02) (0.15s ease)
  - Nav links: opacity increases (0.6 → 1)
```
